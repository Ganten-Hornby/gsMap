Installation guide
==================


The ``GPS`` package can be installed via pip:

it can be installed via pip:

.. code-block:: bash
    :linenos:

    pip install GPS

or via conda:

.. code-block:: bash
    :linenos:

    conda install -c bioconda GPS

or via the source code:

.. code-block:: bash
    :linenos:

    git clone https://github.com/LeonSong1995/GPS.git
    cd GPS
    pip install -e .
